# 1. COLETILLAS

Coletilla (EN) ¿Qué significa? (ES) ¿Para qué se usa en el reading?
As indicated in the text, … Como se indica en el texto…
Para introducir un dato literal o muy claro del texto.
According to the text, … Según el texto… Para responder directamente con información del texto (la más típica).
Based on the text, … Basado en el texto… Para respuestas que implican deducir/explicar a partir del texto, sin inventar.
As explained by the (reading/text), …
Como explica la lectura/el texto…
Para explicar una idea del texto con tus palabras (paráfrasis).
The text indicates that … El texto indica que… Para señalar una conclusión clara del texto.
In the text, it says that … En el texto, se dice que…
Para “citar” la idea sin copiar frases exactas.
According to the text/reading/passage, …
Según el texto/lectura/pasaje…
Igual que “According to the text”, pero variando “text/reading/passage”.
As maintained by the text, …
Como sostiene el texto…
Cuando el texto afirma/defiende una idea (más formal).
As reported by the text, … Como informa el texto…
Estilo “informe”: ideal para hechos/actuaciones.
From my point of view, these words mean …
Desde mi punto de vista, estas palabras significan…
Para definir vocabulario (si el ejercicio lo pide).
What these words imply is …
Lo que estas palabras implican es…
Para explicar el sentido/idea de una expresión.

### Después De Las Coletillas:

Función Plantilla (EN) para pegar tras la coletilla
Cómo usarla (ES) Ejemplo B1 (EN)
Causa → resultado
This happened because + causa, so + resultado.
1) Di por qué pasó. 2) Di qué consecuencia tuvo.
This happened because they left in a hurry, so the guard decided to follow them through the centre.
Orden de hechos
First, … then, … finally, …
Ideal cuando el texto cuenta pasos/acciones. Usa 3 mini-hechos en orden.
First the guard noticed them, then another guard stopped them, and finally the police arrived and checked the situation.
Problema → explicación
The issue was that + problema. This meant that + efecto.
Perfecta para preguntas tipo “what caused… / what was the consequence…”.
The issue was that their stories were different. This meant that the police were unsure who was telling the truth.
Duda / cautela It seems that … because …
Para no “inventar”: describes una impresión basada en un dato del texto.
It seems that the accusation was unclear at first because the boy said he already had the hat when he entered.
Contraste (pero…)
… However, … Úsala cuando hay dos partes opuestas (acusación vs explicación; calma vs tensión).
They said they did not steal it. However, the guard claimed the hat belonged to the shop.
Concesión (aunque…)
Although + idea 1, + idea 2.
Para sonar natural en B1: “aunque X, pasó Y”.
Although they defended their version, the situation became tense and the officers escorted everyone out.
Acción policial + propósito
The officers + Past Simple + to + objetivo.
Muy “policial”: acción + para qué. Ojo: to + infinitive.
The officers checked the hat to confirm the facts and avoid a bigger argument in the backroom.

<!-- Página 4 -->

Prueba / detalle clave
A key detail is that … which + impacto.
Para preguntas que piden “important observation / impact”.
A key detail is that the boys were wearing the hat when they entered, which weakened the accusation of theft.
Habla/declara (reported speech)
The guard claimed that … / The boy insisted that …
Sirve para resumir declaraciones sin copiar comillas.
The guard claimed that they stole the hat, but the boy insisted that it was his and he had it before entering.
Resultado final As a result, … Para cerrar con consecuencia clara.
As a result, the assistant manager’s comment changed the situation and the officers reconsidered the accusation.
